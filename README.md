 # OOPS Project – Registration & Login System

This is an Object-Oriented Python project which contains:

- User Registration System
- User Login System
- Menu-driven App (OOP Based)
- File separation using modules (register.py, login.py, oop_pro.py)
- Uses in-memory database (Python dictionary)

---

## Features

### 1. Registration System
- User can create an account
- Saves name, email, and password
- Prevents duplicate emails
- Stores data in `database` dictionary

###  2. Login System
- Allows registered users to login
- Validates email and password
- Re-prompts if password incorrect
- Shows "User not found" if email doesn't exist

###  3. OOP Structure
- `RegisterSystem` → Handles registration  
- `LoginSystem` → Handles login  
- `NLPApp` (Main class) → Controls menu & app flow
